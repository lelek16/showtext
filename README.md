#ShowText
Few fade animations (via CSS3 transitions) after page load

##Getting Started

### 1. Include js on your site.
```html
<script src="lib/jquery.showtext.min.js"></script>
```
### 2. Initialize
```html
<script>
   $('p').showText(); 
</script>
```


### 3. Options
#### type
`type` type of animation
#### time
`time` time of animation
#### timeout
`timeout` timeout of animation
